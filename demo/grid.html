The Grid

The Holy Kit includes a very powerful grid system, with support for any number of columns, fixed or fluid gutters, and a very easy mechanism to handle the behavior at various resolutions.

The examples are based on a 12 columns grid with a gutter of 20px.

Semantic Grid

By using the .grid-item() mixin you can keep the HTML document free from non-semantic classes (like grid12, grid6 etc...).

Article 1

header { .grid-item(8); }
aside { .grid-item(4); }

Article 2

header { .grid-item(8); }
aside { .grid-item(4); }
